# Этап 1
1. Создайте простой скрипт на выбранной платформе. При запуске приложение должно
подключаться к базе данных и выводить в консоль список всех зарегистрированных
пользователей с их правами (запрос был разработан в задании 1). На данном этапе
подключать веб-сервер не нужно.
2. Создайте репозиторий Git и опубликуйте получившийся проект.
3. Добавьте в проект директорию mysql, сохраните в нее дамп базы в формате SQL и
ER-модель для MySQL Workbench в формате .mwb. Зафиксируйте и отправьте
изменения.
Сдаем этап
# Этап 2
4. Добавьте функционал веб-сервера. Убедитесь, что он работает (создайте простой роут,
который должен открываться в браузере). Зафиксируйте и отправьте изменения.
5. Создайте страницу авторизации (поля логин и пароль). Зафиксируйте и отправьте
изменения.
6. Создайте роут, который принимает данные из формы авторизации и выводит их в
консоль. Зафиксируйте и отправьте изменения.
7. Обработайте запрос отправки формы и верните данные на форму (выведите ошибку
либо сообщите об успехе при авторизации).
Сдаем этап
